NuN Vault

NuN Vault is a secure, immutable event ledger and metamorphosis backup for global event recording and personal restoration.

  • Global, decentralized event storage
  • Encrypted backups for data and metamorphosis states
  • Restore point/rollback for user history
  • API for integration with other NuN modules
  • Secure cloud and atmospheric storage (future)

Status: In Progress – Beta available
